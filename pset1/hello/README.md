# Hello
The first CS50x program written in C to print the name entered by user.

See more instructions: [here](https://cs50.harvard.edu/x/2020/psets/1/hello/)

---
## How to Compile
    $ make hello


## Example
Here is how the program might if the user input is "John".

    $ ./hello
    What is your name?
    John
    hello, John
